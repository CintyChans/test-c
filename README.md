# HELLO ACTION CPP

基于github action，进行cpp的CI、CD测试


## add submodule
- git submodule add moduleA.git moduleA
- git status
- git add .
- git commit -m "add submodule"
- git push 

## rm submodule
- git rm --cached moduleA
- rm -rf moduleA
- rm .gitmodules
- vim .git/config
- git add .
- git commit -m "remove submodule"
- git push
